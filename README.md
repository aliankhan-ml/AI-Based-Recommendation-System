# AI-Based Image-Text Recommendation System

Fine-tuning OpenAI CLIP (ViT-B/32) on a custom 30,832 product image dataset for multimodal similarity search, served via a Flask API and React frontend.

## Results
| Metric | Score |
|---|---|
| Train Accuracy | 99.92% |
| Validation Accuracy | 80.44% |
| Test Accuracy | 80% |
